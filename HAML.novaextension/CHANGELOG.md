# Changelog

All notable changes to this extension are documented here.

## 1.0 — Initial release

- Tree-sitter based syntax highlighting for HAML templates
- Tags, classes, IDs, attribute hashes, doctypes, comments, and filters
- Embedded Ruby highlighted via injection into Nova's built-in Ruby grammar
- Ruby interpolation in text and attributes
- `⌘/` comment toggling using HAML's silent `-#` comment
